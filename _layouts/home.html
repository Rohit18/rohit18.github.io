Urban Flooding

A NASA Terrestrial Hydrology Project

We are trying to address issues related to urban flooding because it is more difficult to map these floods due to the complex and heterogeneous urban landscape. For this purpose, we built a hand-labeled dataset using PlanetScope to capture urban flooding in 14 cities across the world called the Urban Flood Dataset (maybe we will come up with a better name for it).

A sample from our training dataset built using Labelbox. Source: Planet.

Check out my talk on Youtube!

Informal Roads in Central America

A NASA LCLUC Project. Working with and learning a lot from Nicholas Magliocca, Matthew Fagan, Steven Sesnie, Erik Nielsen, Jennifer Devine, and Kendra McSweeney.

We study how narco-trafficking drives Central American deforestation, using informal roads as indicators. We created a locally-trained deep learning model that outperforms Microsoft and OSM in identifying roads. To examine road growth since 2001, we're using Landsat 7 and combining other data sources to build a counter-factual model assessing narco-trafficking's impact on deforestation.

Markers of narco-trafficking in Central America. Sources: Planet and ESA.

Global Surface Water Mapping

A NASA ACCESS Project. Enjoying working with Fritz Pollicelli.

We need validation datasets to evaluate the hundreds (thousands?) of surface water mapping methods using satellite imagery. We built a globally sampled hand-labeled dataset using PlanetScope imagery. Currently building yet another deep learning model with a focus on sampling rather than the method with the help of the impressive SWORD dataset.

Our 90 globally distributed samples overlayed on the different biomes. Source: World Wildlife Fund (WWF) Global Ecoregions Map

Desert Floods in Arizona

A Pima County Flood (Arizona) project.

Keeping it local. We tried to map the fast moving floods in the Sonoran desert and found how difficult it is. Instead of surface water, we identified wet soils after rainfall events using PlanetScope imagery to measure flood extent. We examined the locations of road closures and identified locations where there should be road closures next time there was flooding.

Identifying regions where it flooded, but there weren't any road closures for safety. Source: Planet.

Assessing Intervention Programs by USAID

A USAID’s Bureau for Humanitarian Assistance (BHA). Working with UoArizona's Arizona Institute for Resilience (AIR), Economics, Sociology Departments.

We are assessing the impact of activties performed by local NGOs through the programs funded by BHA that bring about behavorial and landscape changes. Designing a remote sensing based pilot study to measure the impact.

Tools

Geospatial Technologies: I work with geospatial python libraries like rasterio, geopandas, and geemap all the time. GDAL is my favorite (huge shout out to Spatial Thoughts by Ujaval Gandhi). When I need to visualize, I use QGIS, Google Earth Engine, and Microsoft Planetary. To download spatial data, I use GEE, NASA CMR, and Hyp3 Working with geospatial data means that I have to troubleshoot with almost every new dataset I work with, but I have those in our lab's private repos for now. Nothing works out of the box so I have incredible respect for folks who build products with geospatial data.

Machine Learning Tools: I learned deep learning using Keras and Tensforflow (with help from Andrew Ng and Andrej Karpathy's materials) many years ago, but Fastai with Pytorch quickly became my preferred library due to their simpicity. Thank you, Jeremy Howard and Rachel Thomas.

Development Environments: I love coding with vscode (with the help of Copilot-X, which helps me try new things) and sometimes use Jupyter Notebooks (mostly to communicate findigs).

Computational Platforms: When training larger models, I use Google Colab (initial trial runs), Lambdabox (our lab has 2 3080s), High-Performance Computing (HPC) from UofA, and occasionally Alaska Science Facility's OpenScienceLab.

Image Annotation Tools: I spent some time annotating images using Labelbox and NASA ImageLabeler. Labelbox has more features but NASA ImageLabeler is quite impressive in itself.
